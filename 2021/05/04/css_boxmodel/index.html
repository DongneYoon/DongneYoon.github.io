CSS 박스 모델의 속성 정리


대부분의 속성은 숙지하고 있어야 겠지만 개발 시 기억이 안날 수 있으므로 정리 해 둔다.

박스 모델 속성

속성 설명 예시
width 박스 모델의 너비를 지정 width: 100px
height 박스 모델의 높이를 지정 height: 100px
box-sizing 박스 모델의 크기를 게산하는 기준을 지정 box-sizing : border-box / box-sizing : content-box
box-shadow 박스 모델에 그림자 효과를 추가 box-shadow : 가로 세로 넓이 색상

테두리 속성

속성 설명 예시
border-style 상하좌우 4개 방향의 테두리 스타일을 한꺼번에 지정 border-style: solid
border-위치-style 선택한 위치의 테두리 스타일만 지정 border-top-style: solid
border-width 상하좌우 4개 방향 테두리 두께를 한꺼번에 지정 border-width: 10px
border-위치-width 선택한 위치의 테두리 두께만 지정 border-top-width: 10px
border-color 상하좌우 4개 방향의 테두리 색상을 한꺼번에 지정 border-color: #000
border-위치-color 선택한 위치의 테두리 색상만 지정 border-top-color: #f00
border-radius 상하좌우 4개 방향 꼭지점을 한꺼번에 둥글게 지정 border-radius: 10px
border-위치-radius 선택한 위치의 꼭지점만 둥글게 지정 border-top=left-radius: 10px

여백 속성

속성 설명 예시
margin 마진을 한꺼번에 지정 margin: 10px 10px 5px 5px
margin-위치 정해진 위치(top,left,bottom,right)의 마진을 지정 margin-top: 10px
padding 패딩을 한꺼번에 지정 padding: 10px 10px 5px 5px
padding-위치 정해진 위치(top,left,bottom,right)의 패딩을 지정 padding-top: 10px

레이아웃 속성

속성 설명 예시
display 화면에서 요소를 배치하는 방법 지정 (block, inline, inline-block, none ) display : block
float 웹 요소를 왼쪽이나 오른쪽으로 배치. float : left
clear float 속성 해제 clear : both

위치 지정 속성

속성 설명 예시
left, right, top, bottom 기준 위치와 요소 사이에 상하좌우 얼마나 떨어져 있는지 지정 left:10px
position 웹 요소의 위치 지정 (static, relative, absolute, fixed) position: relative
